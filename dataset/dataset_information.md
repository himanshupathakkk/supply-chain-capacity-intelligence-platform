# Dataset Information

## Dataset Source

Brazilian E-Commerce Public Dataset by Olist

Source:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

---

## Dataset Description

This dataset contains operational e-commerce data related to:
- customer orders
- delivery timelines
- seller performance
- freight costs
- logistics operations
- regional customer activity

The dataset was used to build:
- SLA performance KPIs
- operational intelligence dashboards
- freight analysis models
- logistics risk monitoring systems

---

## Key Files Used

- customers_dataset.csv
- orders_dataset.csv
- order_items_dataset.csv
- sellers_dataset.csv

---

## Reason for Excluding Raw Files

Raw datasets were excluded from this repository to:
- improve repository performance
- reduce storage overhead
- maintain cleaner portfolio structure

The complete dataset can be downloaded directly from the Kaggle source above.
